There are many types of viruses that have developed as technology has developed. The one recognized as the first was called Creeper. Written by Bob Thomas in 1971, it spread through DEC PDP-10 computers on ARPAnet and would display the message "I'm the creeper, catch me if you can!" Yet, the first virus detected in the wild was written in 1982 by a 15-year-old in Pittsburgh, Rich Skrenta. It was a boot sector virus called Elk Cloner designed to infect Apply II computers. However, the first PC virus to be found in the wild was Brain, it was created in Pakistan in 1986.

Given some history about how viruses began, this site provides information on three specific types:

Resident Virus:
is a virus that will install itself on a computer allowing it to spread even
when the original "infected" file has been removed.

Direct Action Virus:
is a virus that will attached itself to a specific type of file. When the
file is used, it comes to life and begins searching for similar files to spread to.

Boot Sector Virus:
is a virus that infects the master boot record. It usually spreads by removable media like USB drives
and email attachments.
